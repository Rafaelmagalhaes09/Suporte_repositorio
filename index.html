Pergunta 2: O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar a aparência das páginas HTML, incluindo layout, cores e tipografia.

Pergunta 3: Qual é a diferença entre HTML e CSS?

HTML é usado para criar a estrutura e o conteúdo das páginas, enquanto o CSS é usado para estilizar e formatar essas páginas.
